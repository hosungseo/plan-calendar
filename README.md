# 국가 법정계획 현황판

**https://hosungseo.github.io/plan-calendar/**

국가 법정계획(부처 장관 수립 기본계획) 714건의 수립 시기·주기·심의체계 현황을
2025년 12월 기준 전수 데이터로 보여주는 정적 사이트입니다. 정부 임기와 함께 표시합니다.

- 대통령 임기와 수립 주기 일치: **154건 (21.6%)**
- 임기(2025.6~2030.6) 중 연도별 수립: **매년 104~143건**
- 최근 10년(2016~2025) 신설: **243건** (전체의 34%)
- 시행계획 규정 541건 중 실적 평가 규정은 **222건**
- 심의 위원회: **343종** (295종은 단일 계획만 심의)

## 구성

1. **연도별 수립 현황** — 연도별 수립 예정 계획 수 × 정부 임기 밴드, 연도 드릴다운
2. **계획 신설 추이 (1961~2026)** — 법률에 계획 근거가 담긴 연도별 신설 건수
3. **수립 주기 분포**
4. **수립 절차와 사후 환류** — 절차 5종·시행계획·실적평가·국회보고 규정 현황
5. **심의 위원회 현황**
6. **부처별 수립 연도 분포** — 부처 × 연도 히트맵
7. **전체 계획 목록** — 검색·필터 테이블

## 데이터·방법

- 원자료: `data/raw/national-legal-plans-2025-12.xlsx` (부처 장관이 수립하는 법정 기본계획, 2025.12 기준 코딩)
- 빌드: `python3 scripts/build_data.py` → `data.js`
- "다음 수립" 연도는 최근 수립연도 + 주기의 투영 추정값 (법 개정·계획 변경 시 실제와 다를 수 있음)
- 법률상 확인이 안 되는 항목은 원자료의 공란(미확인)을 그대로 반영 (주기 미확인 52건, 수립연도 미확인 122건)
- 시기 불일치는 이 데이터로 직접 확인되지만, 계획 간 *내용* 중복은 위원회 공유 등 간접 신호로만 볼 수 있음
