# 엇갈린 시계 — 국가 법정계획 714개와 정부 임기

**https://hosungseo.github.io/plan-calendar/**

정부는 5년마다 바뀝니다. 법률이 수립을 명령한 국가계획(법정 기본계획) 714개는 제각각의 주기로 돕니다.

- 대통령 임기와 맞물려 도는 계획: **154개 (21.6%)**
- 새 정부 임기(2025.6~2030.6) 중 매년 **104~143개**가 각자의 주기로 수립
- 계획 심의 위원회 **343종** — 그중 295종은 단 하나의 계획만 심의

이 사이트는 2025년 12월 기준 국가법정계획 현황 전수 코딩본(714건)으로
**계획 수립 시기와 정부 임기의 구조적 불일치**, 그리고 **조정체계의 난립**을 보여줍니다.

## 구성

1. **임기의 시간, 계획의 시간** — 연도별 수립 예정 계획 수 × 정부 임기 밴드, 연도별 드릴다운
2. **같은 5년인데 왜 어긋나는가** — 주기 분포와 위상 불일치
3. **343개의 조정판** — 위원회별 심의 계획 수
4. **어느 부처의 시계가 언제 울리나** — 부처 × 연도 히트맵
5. **714개 전수 탐색** — 검색·필터 테이블

## 데이터·방법

- 원자료: `data/raw/national-legal-plans-2025-12.xlsx` (부처 장관이 수립하는 법정 기본계획, 2025.12 기준 코딩)
- 빌드: `python3 scripts/build_data.py` → `data.js`
- "다음 수립" 연도는 최근 수립연도 + 주기의 투영 추정값 (법 개정·계획 변경 시 실제와 다를 수 있음)
- 법률상 확인이 안 되는 항목은 원자료의 공란(미확인)을 그대로 반영 (주기 미확인 52건, 수립연도 미확인 122건)
- 시기 불일치는 이 데이터로 직접 확인되지만, 계획 간 *내용* 중복은 위원회 공유 등 간접 신호로만 볼 수 있음
